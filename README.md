# Finance Tracker

A full-stack personal finance management application built with .NET Core and Angular.

## Features
- Add, view, and delete financial transactions
- Track income and expenses
- Real-time UI updates
- RESTful API architecture
- In-memory database (easily switchable to SQL Server)

## Tech Stack

**Backend:**
- .NET Core 2.1 Web API
- Entity Framework Core
- Swagger/OpenAPI documentation
- RESTful API design

**Frontend:**
- Angular 21
- TypeScript
- Reactive Forms
- HTTP Client for API communication

## Purpose
Portfolio project demonstrating full-stack development skills with .NET and Angular, including:
- API design and implementation
- Database modeling with EF Core
- Dependency injection
- CORS configuration
- Modern frontend development with Angular standalone components
- End-to-end integration between backend and frontend
